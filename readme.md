# Data Base

## Table name : macchine usate in vendita

## Table columns:

- id | BIGINT | NOTNULL
- marca | VARCHAR(255) | NOTNULL
- modello | VARCHAR(255) | NOTNULL
- categoria | VARCHAR(255) | NULL
- carburante | VARCHAR(255) | NOTNULL
- prezzo | DECIMAL(8,2) | NULL
- chilometraggio | INT | NULL
- anno | YEAR NULL
